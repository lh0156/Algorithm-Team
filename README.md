# 폴더 구조
* 사이트 폴더 - 분류 폴더 - 문제 폴더 - 풀이방법 파일
  * 예) basic1 - 200 - 10828 - SsangYoung.java
* 사이트 폴더: 코드 플러스 강의명으로 작성
  * 예) basic1, intermediate2
* 분류 폴더: 포함되어 있는 문제들을 분류하는 숫자로 작성
  * 예) 200, 1234
* 문제 폴더: 문제 고유 번호로 작성
  * 예) 10828, 3522
* 소스코드: 파스칼 표기법으로 자신의 이름 뒤에 .확장자명을 붙여 작성
  * 예) YouMe.java, HyeIn.java
<br/>
